## Initial setup

1. Locally clone your forked version to begin working on the project.

```bash
git clone git@github.com:Ginohmk/Servo-Life.git
cd Servo-Life.git
```

2. Run

```bash
npm install
```

3. Run

```bash
npm watch
```

4. Open in live server
